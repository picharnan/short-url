# Short Url

This project is just for hobby. If you want to develop just fork. Thanks for your interested.

## Stack

- Vue.js as Frontend
- Nest.js as Backend
- PostgreSQL as Database

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
